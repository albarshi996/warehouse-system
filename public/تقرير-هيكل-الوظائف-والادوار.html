1. نظرة عامة على الهيكل التنظيمي

يمثل الهيكل التنظيمي لمستودعات Brandzo نموذجاً هرمياً من ثلاث طبقات يضمن توزيع واضح للمسؤوليات وسلاسة خطوط الإبلاغ. يضم الهيكل حوالي 320 موظفاً موزعين على 6 إدارات تشغيلية تحت إشراف مدير المستودع التنفيذي.

مع تكامل Odoo 17 Enterprise، كل منصب في الهيكل له دور محدد في النظام مع صلاحيات واضحة على الوحدات المسموح بها، مما يضمن الانتقال المنظم من الهيكل الحالي إلى النظام الرقمي المتكامل.

الهدف الرئيسي: تحويل الهيكل التنظيمي الورقي إلى هيكل رقمي متكامل في Odoo مع تعيين الأدوار والصلاحيات المناسبة لكل مستخدم.

4. الإدارات الستة التشغيلية

الإدارات التشغيلية والتسلسل الهرمي الداخلي
الإدارة المدير المباشر عدد الموظفين التسلسل الهرمي الداخلي الوثائق التي تنتجها
إدارة الاستلام والرقابة مدير قسم الاستلام ~30 موظف مدير ← مشرف ← فريق نماذج GRN، تقارير الرقابة
إدارة فحص ومراقبة الجودة مدير إدارة الجودة ~15 مفتش مدير ← فرق الفحص تقارير الجودة، شهادات المطابقة
إدارة التخزين وتحديد المواقع مدير عمليات التخزين ~70 موظف مدير ← مهندس ← مشرفي ← فرق خرائط المواقع، تقارير الجرد
إدارة تحضير الطلبات والتوزيع مدير عمليات التوزيع ~120 عامل مدير ← مشرفي ← مناطق ← عمال قوائم السحب، إذنات التسليم
إدارة أسطول النقل والتوزيع مدير النقل والتوزيع ~70 سائق مدير ← منسق ← فرق جداول النقل، تقارير الأسطول
إدارة المرتجعات وتصحيح المخزون مدير إدارة المرتجعات ~12 موظف مدير ← مشرف ← فرق نماذج الإرجاع، سندات التسوية

شرح المصفوفة:
  • R = Responsible (مسؤول رئيسي)
  • A = Accountable (مسؤول عن التنفيذ)
  • C = Consulted (يستشار)
  • I = Informed (يُعلم)

6. جدول تعيين الأدوار في أودو

جدول تعيين كامل: المنصب → Role في أودو → الوحدات → مستوى الصلاحية
المنصب Role في أودو الوحدات المسموح بها مستوى الصلاحية
مدير المستودع التنفيذي Inventory Manager كل وحدات المخزون Admin
مدير المستودع التنفيذي System Administrator Settings كاملة Admin
مدير تقنية المعلومات System Administrator Settings كاملة Admin
مدير الجودة والسلامة الغذائية Quality Administrator Quality + Inventory (read) Admin
مدير المحاسبة والرقابة المالية Accounting Manager Purchase + Accounting Manager
مشرف الاستلام Inventory User Inventory + Quality User
مشرف التخزين Inventory User Inventory + Barcode User
مشرف الجودة Quality User Quality + Inventory (read) User
عامل السحب Barcode User Barcode فقط User
السائق Fleet User Fleet User
مفتش الجودة Quality User Quality + Inventory (read) User
ملاحظة هامة: هذا الجدول يضمن تعيين الأدوار المناسبة لكل مستخدم في Odoo مع الصلاحيات المناسبة، مما يضمن الأمان والامتثال في النظام.
